Définition

Le code HTML permet de créer des blocs de code avec des fonctions particulières

Le code html est constitué de blos de code encadré par des balises, il existe deux types de balises:

  1. Les balises encadrant du code, la balise fermente commence par < /balise>
  2. Les balises orphelines sont des balises qui n'encadrent aucun code et elles se ferment elle-mêmes. Par exemple <br/> <hr/>
br = break row
h = header
hr = ....row
